# Last War Mini

광고에서 자주 보이는 "라스트워" 스타일 미니게임. 부대를 좌우로 움직이며 자동 사격으로 좀비를 막아내고, 게이트를 골라 병력을 불려서 레벨을 하나씩 깨나가는 게임입니다.

## 실행 방법

별도 빌드 없이 브라우저에서 바로 실행됩니다.

```bash
open index.html
```

또는 로컬 서버로:

```bash
npx serve .
# 혹은
python3 -m http.server 8000
```

## 조작

- **마우스/터치 드래그** 또는 **← → (A/D) 키**: 좌우 이동
- 사격은 자동입니다.

## 게임 규칙

- 좀비가 방어선(파란 선)에 닿으면 병력이 줄어들고, 병력이 0이 되면 게임 오버.
- **게이트**: 좌/우 둘 중 하나를 골라 통과합니다.
  - 파란 게이트(`+N`, `x2`): 병력 증가
  - 빨간 게이트(`-N`): 병력 감소 (총으로 쏘면 패널티가 줄어듭니다!)
  - `+` 게이트도 총으로 쏘면 수치가 올라갑니다.
- **아이템** (좀비 처치 시 드랍):
  - 노랑(연사): 8초간 발사 속도 2배
  - 주황(공격): 8초간 공격력 2배
  - 초록(+5): 병력 +5
  - 빨강(핵): 화면의 적 전멸 (보스는 큰 피해)
- 일정 시간 버티면 **보스**가 등장하고, 보스를 잡으면 레벨 클리어 (보너스 병력 +5).
- 레벨이 오를수록 적의 체력/속도/물량이 늘어납니다.

## 적 종류

| 적 | 특징 |
|---|---|
| 좀비 | 기본형 |
| 러너 | 빠르지만 약함 |
| 탱커 | 느리지만 단단함 (레벨 2부터) |
| 보스 | 레벨 마지막 등장, 방어선에 닿으면 즉시 게임 오버 |
